BiReader is an online application that supports bilingual reading in Python using a Postgres database. Bilingual reading is a practice of reading the same book in 2 languages. Users can log in and upload pairs of machine-readable PDFs of the same book in two different languages. The backend then processes the PDFs, performs sentence tokenization using Spacy, and then uploads them to the server. The users are then able to choose a pair of books that they uploaded and read them side by side.
